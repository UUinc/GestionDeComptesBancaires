# GestionDeComptesBancaires
On se propose de gérer les comptes bancaires d’une agence donnée.
les opérations :
  - Ajout Client
  - Creation Compte
  - Retrait
  - Depot
  - Virement
  - Supprimer Client
  - Cloturer Compte
